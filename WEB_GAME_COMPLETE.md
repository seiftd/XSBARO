## 🚀 Quick Start Commands

### Launch the Game
```bash
# Make executable (first time only)
chmod +x launch_web_game.sh

# Launch the web game
./launch_web_game.sh
```

### Manual Launch
```bash
# Using Python 3
cd public && python3 -m http.server 8080

# Using Node.js
cd public && npx http-server -p 8080

# Then open: http://localhost:8080/game.html
```
